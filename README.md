# :inbox_tray: Como baixar o projeto
```bash
	
	# Clonar o repositório
    $ git clone https://github.com/josethome/sistema-vendas-csharp
    
    # Entrar no diretório
    $ cd sistema-vendas-csharp

    # Instalar as dependências
    $ yarn install

    # Iniciar o projeto
    $ yarn start
    
```
